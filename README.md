# Website Portfolio
This is my personal website portfolio that I worked on after graduating from university. It was my very first time learning and attempting frontend. As you read this, I would have ventured even further into frontend,  taking all the lessons that I have learnt from this project. Albeit simple, I am very proud of this website and I hope that you enjoy looking at it too. 

## How to Run
1. Download or clone the repo
2. Go into `html_things` and open `homepage.html` in your browser 

Please note that this was designed and optimised for large screens only.

## What I have Learnt
Self-studying HTML, CSS, and JavaScript on YouTube beforehand seemed simple at first. However, when it came to actual implementation, everything became very confusing very fast. For example, it took me weeks to understand exactly how to use grids and flexboxes. It subsequently took me even more time in understanding nested grids and flexboxes. 

It was only through the implementation of this project that I learnt what responsive design meant. When I had implemented this, my skills and knowledge were inadequate to reach an acceptable-level of responsive design. Thus, it had only been optimised for large screens. In the future, I aim to modify this project of mine to fit screens of all sizes.
